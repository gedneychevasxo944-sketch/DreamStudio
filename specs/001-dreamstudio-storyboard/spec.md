# Feature Specification: DreamStudio 2.0 故事板创作系统

**Feature Branch**: `feature/storyboard-v2`
**Created**: 2026-04-17
**Updated**: 2026-04-21
**Status**: In Review
**Input**: DreamStudio 2.0 产品方案 —— 最终交付版 + 产品路径讨论

## 1. 概述

DreamStudio 2.0 是一个AI驱动的结构化创作工具，支持用户从剧本到视频的完整管线，同时兼顾快速探索与深度定制。

**核心设计原则**：
- 对话是全局的，随叫随到
- 复杂需求用画布，简单需求用对话
- 资产绑定带版本的工作流模板
- 主页只做介绍，创作统一在故事板

### 2.2 对话助手定位

对话助手是**全局浮窗能力**，默认悬浮在任何界面上：
- 位置：默认在右下角，可自由拖动
- 大小：可调整
- 吸附：默认不吸附边缘，用户可选择吸附到某个边缘
- 关闭：缩小为顶部导航栏的图标按钮，不阻碍任何创作区域
- 重新打开：点击导航栏图标恢复浮窗

**浮窗规则**：
- 故事板：锚定到当前选中的资产
- 画布：锚定到当前选中的节点或工作流全局
- 不选择任何资产/节点：项目级/工作流级对话

## 3. 进入流程

### 3.1 主页

主页作为品牌展示和案例展示，不承载直接创作功能。

**页面结构**：
- 右上角：登录/注册按钮（未登录）或用户头像+昵称+退出按钮（已登录）
- Hero 区域：Logo "造梦AI" + 标语 + "开始创作"按钮 + "使用文档"按钮
- 精选案例：横向滚动卡片（封面图、时长、浏览量、标题、描述、标签）
- 创作流程：5步流程展示（剧本→角色→场景→分镜→视频）
- 产品特性：4个核心特性展示

**验收场景**：
1. **Given** 用户打开首页，**When** 查看页面，**Then** 看到品牌展示、精选案例、创作流程说明
2. **Given** 用户点击案例卡片，**Then** 弹出视频预览窗口
3. **Given** 用户在预览窗口点击"进入工作台"，**Then** 进入创作流程（需先登录）
4. **Given** 用户点击"使用文档"，**Then** 滚动到创作流程区域

### 3.2 开始创作入口

点击"开始创作"后的流程：

**验收场景**：
1. **Given** 用户点击"开始创作"，**When** 未登录，**Then** 弹出登录/注册弹窗
2. **Given** 用户登录成功，**When** 点击"开始创作"，**Then** 先加载项目列表
3. **Given** 项目列表加载完成，**When** 没有历史项目，**Then** 直接进入空白故事板引导（不弹窗）
4. **Given** 项目列表加载完成，**When** 有历史项目，**Then** 显示项目选择对话框

### 3.2.1 项目选择对话框

项目选择对话框用于选择已有项目或创建新项目：

**对话框规格**：
- 尺寸：固定宽度 560px，高度 520px（不变）
- 位置：屏幕居中，遮罩层背景
- 动画：淡入淡出（opacity），无缩放动画（避免与 CSS transform 冲突）

**四种状态**：

| 状态 | 条件 | 展示 |
|------|------|------|
| 创建状态 | 点击"创建新项目" | 显示输入框和确认/取消按钮 |
| 加载状态 | `loading=true` | 显示骨架屏（2个占位块） |
| 有项目状态 | `projects.length > 0` | 显示搜索框 + 项目列表 |
| 空状态 | `projects.length === 0` 且无搜索词 | 显示"还没有任何项目"提示 |

**有项目状态交互**：
- 顶部显示搜索框，支持按项目名称过滤
- 搜索无结果时显示"没有找到匹配的项目"
- 项目列表可滚动（固定高度）
- 点击项目进入该项目
- 点击"创建新项目"切换到创建状态

**验收场景**：
1. **Given** 显示项目列表，**When** 用户输入搜索词，**Then** 实时过滤显示匹配项目
2. **Given** 项目列表滚动到底部，**Then** 继续滚动不受影响（列表独立滚动）
3. **Given** 用户点击遮罩层，**Then** 关闭对话框

### 3.3 空白故事板引导

新项目进入空白故事板时，显示引导选项：

**引导选项**：
- "上传剧本" - 打开上传弹窗，支持文件上传或粘贴文本
- "从空白开始" - 直接进入空白剧本阶段

**AI 生成剧本**：通过顶部导航栏助手发起，不在引导区显示

**验收场景**：
1. **Given** 用户进入空白故事板，**When** 无任何内容，**Then** 显示引导区："你想从哪里开始？"
2. **Given** 引导区显示，**When** 用户选择"上传剧本"，**Then** 打开剧本上传弹窗
3. **Given** 引导区显示，**When** 用户选择"从空白开始"，**Then** 清空引导区，进入空白剧本阶段
4. **Given** 用户在剧本阶段，**When** 想要AI生成剧本，**Then** 点击顶部导航栏助手发起对话

### 3.4 TopBar 项目切换器

TopBar 左侧提供项目名称和版本号下拉切换器，中间提供创作模式切换下拉：

**布局**：
```
[造梦AI] [项目名称] [V1.0▼]  |  [故事板 ▼]  |  [资产] [助手] [保存] [···]
```

**创作模式**：
- 故事板：标准创作流程，适合大多数用户
- 自由创作：适合进阶用户，可自由编辑工作流节点，hover 显示 tooltip "适合进阶用户自由创作"

**交互规则**：
- **点击项目名称** → 进入编辑模式（输入框），Enter 或失焦保存
- **点击 ▼ (项目)** → 显示项目下拉菜单
- **点击 V1.0 ▼** → 显示版本下拉菜单
- **点击故事板 ▼** → 显示模式切换下拉菜单（点击触发，非 hover）

**项目下拉菜单**：
- 第一行：创建故事板按钮（渐变背景，36px 高）
- 第二行：搜索框（32px 高，可过滤项目）
- 下方：项目列表（最大高度 340px，超出可滚动）
- 每个项目显示：文件夹图标 + 名称 + 更新时间 + ✓ 标记（当前项目）
- 暂无历史项目时显示空状态提示

**版本下拉菜单**：
- 三列紧凑布局（勾选 20px | 版本名 flex | 时间 72px）
- 当前版本高亮显示
- 最大高度 300px，超出可滚动
- hover 显示备注 tooltip

**验收场景**：
1. **Given** 用户在故事板，**When** 点击项目名称，**Then** 名称变为输入框可直接编辑
2. **Given** 用户在编辑状态，**When** 按 Enter 或失焦，**Then** 保存新名称
3. **Given** 用户点击 ▼ (项目)，**Then** 显示项目下拉菜单
4. **Given** 下拉菜单显示，**When** 用户选择其他项目，**Then** 切换到该项目
5. **Given** 用户点击 V1.0 ▼，**Then** 显示版本历史列表
6. **Given** 版本列表显示，**When** 用户选择某版本，**Then** 切换到该版本
7. **Given** 用户点击故事板 ▼，**Then** 显示模式切换下拉（故事板/自由创作）
8. **Given** 模式下拉显示，**When** 用户选择自由创作，**Then** 切换到自由创作画布界面

## 4. 用户场景

### 用户场景 1 - 小白用户：快速创作（优先级：P1）

**描述**：用户进入故事板，通过对话引导完成从剧本到视频的完整创作。

**为什么这个优先级**：覆盖80%的目标用户，是产品的核心价值体现。

**独立可测试**：可以通过对话完成一个完整项目：剧本生成 → 角色生成 → 分镜生成 → 视频生成。

**验收场景**：
1. **Given** 用户在故事板，**When** 对话说"帮我生成3个主角"，**Then** AI生成角色候选列表，用户确认后进入角色列表
2. **Given** 用户在视频阶段，**When** 点击某个镜头，**Then** 显示视频播放器
3. **Given** 用户在任意阶段，**When** 对话助手显示"你在和 XXX 对话"，**Then** 用户知道当前上下文

---

### 用户场景 2 - 进阶用户：微调 + 快速素材（优先级：P2）

**描述**：用户在故事板中微调已有资产，必要时使用悬浮助手快速生成素材。

**为什么这个优先级**：满足进阶用户的效率需求。

**独立可测试**：可以选择已有角色，通过对话修改形象（"把头发改成蓝色"）。

**验收场景**：
1. **Given** 用户选中了"红发女"角色，**When** 对话说"把头发改成蓝色"，**Then** 角色形象更新
2. **Given** 用户在故事板，**When** 对话说"画一个蒸汽朋克猫"，**Then** 生成图片，显示"添加到故事板"按钮
3. **Given** 资产修改后被下游引用，**When** 用户修改该资产，**Then** 显示影响提示条

---

### 用户场景 3 - 技术用户：复杂工作流（优先级：P3）

**描述**：用户打开画布，搭建可复用的复杂工作流，保存为模板。

**为什么这个优先级**：满足技术用户的深度定制需求，是产品差异化的体现。

**独立可测试**：可以进入画布，添加多个节点，连接后保存为模板，使用模板生成资产。

**验收场景**：
1. **Given** 用户在故事板，**When** 点击"🔧工作流编辑"，**Then** 进入全屏画布，当前上下文已填充
2. **Given** 用户在画布，**When** 没有节点选中且说"生成一张图"，**Then** 结果回到故事板
3. **Given** 用户在画布，**When** 说"增加一个节点"，**Then** 在画布内添加节点
4. **Given** 用户完成画布操作，**When** 点击"保存"，**Then** 工作流保存，可选择"发布为模板"

---

### 用户场景 4 - 剧本解析（优先级：P1）

**描述**：AI自动识别剧本中的角色、场景、道具，用户确认后生成对应资产。

**验收场景**：
1. **Given** 用户在剧本阶段有剧本内容，**When** 点击"AI解析"，**Then** 显示识别结果列表（角色、场景、道具）
2. **Given** AI显示识别结果，**When** 用户勾选确认，**Then** 对应资产进入相应阶段

---

### 用户场景 5 - 分镜生成（优先级：P1）

**描述**：AI根据剧本、角色、场景、道具自动生成分镜参数和Prompt，用户微调后生成。

**验收场景**：
1. **Given** 用户在分镜阶段，**When** 点击"AI生成分镜"，**Then** 显示分镜列表，包含镜头描述、关联资产、景别、运镜等
2. **Given** 分镜列表显示，**When** 用户点击某个镜头，**Then** 右侧详情面板可微调参数
3. **Given** 用户调整参数后，**When** 点击"生成视频"，**Then** 调用视频生成接口

---

### 用户场景 6 - 上传资源（优先级：P2）

**描述**：用户在各阶段上传已有资源，AI辅助处理或直接展示。

**验收场景**：
1. **Given** 用户在剧本阶段，**When** 点击"上传剧本"，**Then** 支持选择 .txt, .pdf 文件
2. **Given** 用户上传文件后，**When** 系统解析完成，**Then** 显示处理选项："直接展示" 或 "AI处理"
3. **Given** 用户选择"AI处理"，**When** 上传角色图，**Then** AI分析图片生成描述；选择"直接展示"则仅保存图片
4. **Given** 各阶段资产详情面板，**When** 查看资源区，**Then** 显示该阶段所有资源（上传的 + AI生成的）

---

### 用户场景 7 - 对话上传文件（优先级：P2）

**描述**：用户通过对话上传文件作为上下文。

**验收场景**：
1. **Given** 用户在对话助手，**When** 点击上传按钮，**Then** 支持上传图片、视频、文本文件
2. **Given** 用户上传文件，**When** 文件被接收，**Then** 文件作为当前对话的上下文处理

---

### 边界情况

- 当资产没有任何关联时（如新建项目），对话助手显示项目级对话
- 当用户切换阶段时，保持上一个阶段的对话历史
- 当工作流执行失败时，显示错误信息并提供重试选项
- 当模板作者更新模板时，链接该模板的资产收到通知
- 当上传文件格式不支持时，显示错误提示

## 5. 功能需求

### 5.1 主页与进入流程

- **FR-001**: 主页作为品牌展示和案例展示，包含精选案例、创作流程说明、产品特性介绍
- **FR-002**: 右上角显示登录/注册按钮（未登录）或用户头像+昵称+退出按钮（已登录）
- **FR-003**: "开始创作"按钮点击后检查登录状态，未登录弹出登录/注册弹窗
- **FR-004**: 已登录用户点击"开始创作"后先加载项目列表
- **FR-005**: 进入故事板时如果没有历史项目，默认创建新项目并进入空白故事板引导（不弹窗）
- **FR-006**: 进入故事板时如果有历史项目，显示项目选择对话框
- **FR-007**: 项目选择对话框支持四种状态：创建状态、加载状态、有项目状态、空状态
- **FR-008**: 项目选择对话框支持搜索过滤项目列表
- **FR-009**: 空白故事板必须显示引导区，引导用户选择起点

### 5.2 项目切换

- **FR-010**: TopBar 左侧显示项目名称和版本号下拉切换器，中间显示创作模式下拉
- **FR-010a**: 点击项目名称可直接编辑，Enter 或失焦保存
- **FR-010b**: 点击 ▼ 展开项目下拉菜单，第一行创建故事板按钮，第二行搜索框，下方项目列表（最大高度 340px）
- **FR-010c**: 点击版本号 ▼ 展开版本下拉菜单，三列紧凑布局（勾选 20px | 版本名 flex | 时间 72px），最大高度 300px
- **FR-011**: 项目下拉包含：当前项目（标记 ✓）、历史项目列表（显示更新时间）、搜索过滤；版本下拉显示当前版本勾选标记和时间
- **FR-012**: 切换项目时，如果有未保存修改，提示用户确认
- **FR-012a**: 切换版本时，无须确认直接切换
- **FR-012b**: 创作模式下拉包含故事板和自由创作两项（点击触发非 hover），点击切换，当前模式高亮；自由创作 hover 显示 tooltip "适合进阶用户自由创作"
- **FR-012c**: 前端实时自动保存当前项目草稿（本地草稿），用户点击保存后再发送给后端
- **FR-012d**: 全局保存按钮位于 TopBar 右侧，各阶段编辑器内不设置保存按钮
- **FR-012e**: 返回主页、切换项目、刷新界面时，如果有未保存修改，弹出确认对话框

- **FR-013**: 对话助手是浮窗形式，默认在右下角，可拖动到任意位置，可调整大小，可选择吸附到边缘。根据当前选中资产/节点自动切换上下文，关闭后显示在顶部导航栏图标
- **FR-014**: 切换到资产时，显示该资产的简介（名称、类型、关联资产），由前端组装，无需调用AI
- **FR-015**: 切换到节点时，对话操作锚定到该节点
- **FR-016**: 没有选中任何对象时，对话为项目级/工作流全局级
- **FR-017**: 对话助手支持上传文件（图片、视频、文本）作为上下文

### 5.4 故事板阶段系统

- **FR-018**: 系统必须支持7个阶段：剧本、角色、场景、道具、分镜、视频、剪辑
- **FR-019**: 每个阶段显示对应的资产类型列表
- **FR-020**: 阶段导航必须显示完成状态（已完成显示✓，当前高亮）
- **FR-021**: 剧本阶段必须支持富文本编辑 + AI生成 + 素材提取
- **FR-021a**: 剧本阶段配备专属AI助手面板（右侧固定30%宽度），支持对话生成剧本，接受/拒绝后写入编辑器
- **FR-021b**: 剧本阶段支持目录导航（手动触发或AI生成后自动生成）
- **FR-021c**: 剧本阶段AI提取素材功能，非阻塞弹窗，勾选确认后批量添加到角色/场景/道具阶段
- **FR-022**: 角色/场景/道具阶段必须支持AI从剧本解析并生成。各阶段拥有独立的AI对话面板（右侧固定展示），支持优化资产描述词，对话后可选择"立即生成图片"或"取消"。每阶段的AI对话状态（选中的资产、对话内容）独立保存，切换阶段时保持状态不丢失。
- **FR-023**: 分镜阶段是视频创作的核心规划阶段。每个分镜是一个镜头（shot），包含：镜头编号、名称、关联剧本段落、关联资产（角色/场景/道具合并一行展示）、镜头参数（风格、运动、景别、时长5-15秒）、镜头Prompt。分镜阶段可生成3张预览帧图片（首帧/关键帧/尾帧），但不直接生成视频。预览帧hover显示缩略图。
- **FR-024**: 视频阶段是视频生成阶段。左侧显示视频片段列表（Prompt资产，按"视频片段+序号"命名），点击后右侧显示详情：关联剧本（单独一行）、关联角色/场景/道具（合并一行，hover查看缩略图）、生成参数（画面比例/时长5-15秒/画质）。用户可编辑Prompt并单独生成视频。视频阶段无批量生成按钮。
- **FR-025**: 剪辑阶段用于将多个视频片段组装成完整成片。提供时间线视图、拖拽排序、裁剪（设置起止时间）、无转场（首尾相接）。最终导出成一个完整的视频文件（可选格式：MP4/MOV/WebM）。

### 5.5 资产详情面板

- **FR-026**: 选中资产后，右侧面板显示详情编辑区
- **FR-027**: 详情面板包含：基本信息、参数配置、预览、对话助手
- **FR-028**: 修改资产后，系统必须提示下游影响范围
- **FR-029**: 详情面板包含资源区，显示该阶段所有资源（上传的 + AI生成的）

### 5.6 上传与AI处理

- **FR-030**: 支持的文件类型：剧本（.txt, .pdf）、图片（角色/场景/道具参考）、视频
- **FR-031**: 上传后必须让用户选择处理方式："直接展示" 或 "AI处理"
- **FR-032**: AI处理根据资产类型不同：剧本→解析提取实体，图片→分析生成描述

### 5.7 快速生成（对话式）

- **FR-033**: 用户通过自然语言描述需求，系统自动理解并执行
- **FR-034**: 系统自动选择合适的模板（系统预设或用户创建的）
- **FR-035**: 生成结果必须经过用户确认后才正式进入故事板
- **FR-036**: 每个回复必须提供"下一步推荐"，根据上下文动态生成
- **FR-037**: 对话过程中，系统可以反问确认用户意图

### 5.8 画布编辑器

- **FR-038**: 画布是全屏工作流编辑器，可从故事板"🔧工作流编辑"进入
- **FR-039**: 画布支持：节点拖拽、节点连线、缩放平移、节点收拢/展开
- **FR-040**: 画布内对话助手锚定规则：
  - 无节点选中：全局对话，操作结果回到故事板
  - 有节点选中：对话锚定到该节点
- **FR-041**: 画布内可以"测试运行"单个节点
- **FR-042**: 正式运行后，结果不自动替换，提供"替换/新增/取消"选项

### 5.9 模板系统

- **FR-043**: 模板是可复用的单节点或多节点工作流
- **FR-044**: 模板有两种来源：系统预设、用户创建
- **FR-045**: 模板有两个触发时机：系统自动判断、用户手动选择
- **FR-046**: 用户可将当前工作流"发布为模板"
- **FR-047**: 模板支持版本管理，修改后产生新版本

### 5.10 资产-工作流关系

- **FR-048**: 每个资产绑定一个带版本的工作流
- **FR-049**: 资产使用模板生成时，记住所使用的模板和版本
- **FR-050**: 当模板新版本发布时，系统通知相关资产用户
- **FR-051**: 替换资产时，如果工作流版本变更，必须提示用户

### 5.11 影响提示

- **FR-052**: 修改剧本后，提示可能影响的角色/场景/道具
- **FR-053**: 修改被分镜引用的资产时，提示需要重新生成的镜头

### 5.12 主题系统

- **FR-054**: 系统支持浅色（day）和深色（night）两种主题模式
- **FR-055**: 主题切换通过 `data-theme` 属性控制，配合 CSS 变量实现
- **FR-056**: 主题偏好保存在 localStorage（key: `dreamstudio-theme`），刷新后保持
- **FR-057**: 主题切换按钮位于主页右上角和故事板顶部导航栏
- **FR-058**: 主题状态通过 `useThemeStore` (Zustand) 统一管理

## 6. 数据模型

### 6.1 项目 (Project)
- `id`: 唯一标识
- `name`: 项目名称
- `stages`: 各阶段的数据引用
- `createdAt`, `updatedAt`

### 6.2 资产 (Asset)
- `id`: 唯一标识
- `projectId`: 所属项目
- `type`: 类型 (script | character | scene | prop | storyboard | video)
- `stage`: 所属阶段
- `name`: 资产名称
- `content`: 内容 (文本/图片URL/视频URL/参数JSON)
- `workflowBinding`: { `workflowId`, `versionId` } — 绑定的工作流和版本
- `createdAt`, `updatedAt`

### 6.3 资源 (Resource)
- `id`: 唯一标识
- `assetId`: 所属资产（可为空，表示未关联）
- `type`: 类型 (upload | ai_generated)
- `fileType`: 文件类型 (image | video | text | pdf)
- `url`: 资源地址
- `aiProcessed`: 是否经过AI处理
- `aiDescription`: AI处理后的描述（如果有）
- `createdAt`

### 6.4 工作流 (Workflow)
- `id`: 唯一标识
- `name`: 工作流名称
- `description`: 描述
- `graph`: { nodes: [], connections: [] } — 节点和连线
- `isTemplate`: 是否为模板
- `isSystem`: 是否为系统预设
- `linkedAssets`: 使用该模板的资产ID列表
- `createdAt`, `updatedAt`

### 6.5 节点 (Node)
- `id`: 唯一标识
- `workflowId`: 所属工作流
- `type`: 节点类型 (textgen | imagegen | videogen | control | transform)
- `position`: { x, y }
- `config`: 节点配置 (模型、提示词、参数等)
- `inputs`, `outputs`: 端口定义

### 6.6 模板版本 (TemplateVersion)
- `id`: 唯一标识
- `workflowId`: 所属工作流
- `version`: 版本号 (v1, v2, v3...)
- `graph`: 该版本的图结构快照
- `createdAt`

### 6.7 对话会话 (ChatSession)
- `id`: 唯一标识
- `assetId` 或 `workflowId` 或 `projectId`: 会话归属
- `messages`: 消息历史 [{ role, content, attachments? }]
- `createdAt`, `updatedAt`

## 11. 资产库

资产库是全局跨项目的资产管理中心，支持从任意项目访问和复用资产。

### 11.1 设计原则

- **项目隔离**：默认只显示当前项目的资产，避免跨项目污染
- **跨项目搜索**：可通过搜索功能跨所有项目查找资产
- **来源无关**：不区分资产来源（对话/故事板/工作流），统一作为素材使用
- **血缘简化**：不保留血缘链路，资产可直接复用

## 13. 节点工作区

节点工作区是画布中选中节点后右侧出现的面板，用于显示和编辑节点的输入输出。

### 13.1 通用布局

```
┌─────────────────────────────────────────────────┐
│ [T] 节点名称           类型         [V0]   🔒   │  ← 节点标题 + 类型 + 版本徽章
├─────────────────────────────────────────────────┤
│  输入                                          │
│ ┌─────────────────────────────────────────────┐│
│ │                                             ││
│ │                                             ││  ← 输入内容区（撑满）
│ └─────────────────────────────────────────────┘│
├─────────────────────────────────────────────────┤  ← 分割线
│  输出                                          │
│ ┌─────────────────────────────────────────────┐│
│ │                                             ││
│ │                                             ││  ← 输出内容区（撑满）
│ │                                             ││
│ └─────────────────────────────────────────────┘│
├─────────────────────────────────────────────────┤
│        [💬 对话]      [📜 运行历史]           │  ← 底部 Tab
└─────────────────────────────────────────────────┘
```

**布局规则**：
- 抽屉关闭时：输入区和输出区各占 50% 高度
- 抽屉展开时：输入/输出区显示 Tab 切换
- 版本号：动态显示 `V0`, `V1`, `V2`... 无下拉选择

### 13.2 Producer 节点

**输入**：
- 无上游：用户输入 textarea（placeholder: "请输入你的想法、创意或剧本大纲..."）
- 有上游：显示上游节点来源 + 上游输出内容（只读）+ 补充输入（可选）

**输出**：
- 预览：显示前 300 字符 + "查看全文"按钮
- 全屏：Markdown 阅读器弹窗

```
┌─────────────────────────────────────────────┐
│  输入                                        │
│ ┌─────────────────────────────────────────┐ │
│ │ 🔗 来自：剧本生成器                      │ │
│ │ ─────────────────────────────────────── │ │
│ │ 上游节点输出内容...                      │ │
│ └─────────────────────────────────────────┘ │
│ ┌─────────────────────────────────────────┐ │
│ │ + 补充输入（可选）                      │ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 13.3 Content 节点

**输入**：
- 无上游：用户输入 textarea
- 有上游：显示上游节点来源 + 上游输出内容（只读）+ 补充输入（可选）

**输出**：
- 预览：显示前 300 字符 + "查看全文"按钮
- 全屏：小说阅读器弹窗（左侧目录 + 右侧章节内容）
